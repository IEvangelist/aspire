## Install the SDK

Install either [.NET 8.0](https://dotnet.microsoft.com/download/dotnet/8.0) or [.NET 9.0](https://dotnet.microsoft.com/download/dotnet/9.0) to work with Aspire. No prior .NET experience is required.

### Install OCI compliant container runtime

Aspire uses an OCI compliant container runtime to run and manage containers. The most common option is Docker Desktop, which provides a user-friendly interface for managing containers and images.

<LinkButton href="https://www.docker.com/products/docker-desktop" icon="seti:docker" iconPlacement="start" variant="secondary">
  Docker Desktop
</LinkButton>

Alternatively, you can use [Podman](https://podman.io/), which is a daemonless container engine that can run containers without requiring root privileges.

<Tabs>
  <TabItem label="Linux" icon="linux">
	  <Code title='Set the ASPIRE_CONTAINER_RUNTIME env var to podman'
          code={`export ASPIRE_CONTAINER_RUNTIME=podman`} lang="bash" />
  </TabItem>

  <TabItem label="Windows" icon="seti:windows">
	  <Code title='Set the ASPIRE_CONTAINER_RUNTIME env var to podman'
          code={`[System.Environment]::SetEnvironmentVariable("ASPIRE_CONTAINER_RUNTIME", "podman", "User")`} lang="powershell" />  
  </TabItem>
</Tabs>
